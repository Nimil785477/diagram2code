# Contributing to diagram2code

Thank you for your interest in this project.

## Current Status

This project is actively developed and currently in a focused development phase:
- Core architecture is still evolving
- Benchmarking and predictor performance are the main priorities
- Major design decisions are still being finalized

## Contribution Policy (Important)

At this stage, external contributions are **not actively accepted**.

This is intentional to:
- maintain architectural consistency
- ensure benchmark reproducibility
- avoid fragmentation during critical development phases

## What You Can Do

You are welcome to:
- open issues for bugs or unexpected behavior
- suggest improvements or ideas
- discuss use cases or real-world applications

## Future Contributions

Once the project reaches a more stable phase, contribution guidelines will be expanded and pull requests will be welcomed.

Thank you for your understanding.
